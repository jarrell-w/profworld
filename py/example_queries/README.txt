--------------------------------
       SUPER COOL QUERIES
           BY DANTE
--------------------------------
List of example queries in this
directory (explanations below):

1. all_problems.txt
2. test_problems.txt
3. user_tests.txt
4. problems_by_tags.txt

--------------------------------

1. all_problems
 - Returns all the problems in
   the database at once

2. test_problems
 - Returns all of the questions
   that correspond to the 
   given test
 - Requires the test_id of the
   test you are searching for

3. user_tests
 - Returns all of the test_ids
   that correspond to the 
   given user
 - Requires the user_id of the
   user you are searching for

4. problems_by_tags
 - Returns all of the problems
   that have the given tags
 - Requires the following
   information (even if NULL)
   - category_tag
   - region_tag
   - treatment_tag
